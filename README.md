# 게시판 API 서버
# 소개
* 게시판 API 서버는 JWT 기반 게시판 기능을 구현한 서버입니다.

# 주요 기능
 * JWT 기반 사용자 인증/인가 제한
 * 게시글 및 댓글 작성/조회/수정/삭제
 * 게시글 좋아요
 * AWS S3 파일 업로드
  
# API 문서 [링크]
 * Spring RestDocs 기반 API 문서 제공
